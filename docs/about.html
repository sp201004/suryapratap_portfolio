🙋‍♂️ About Me

I’m a Software Engineer Intern at Amdocs and a Computer Science graduate from NIT Agartala. I build cloud-native platforms — the kind that sit between developers and infrastructure and make complicated systems feel simple.

At Amdocs I work on mCloud, an in-house private cloud provisioning platform replacing the legacy VMware vRA stack. The work spans Node.js microservices, async orchestration through RabbitMQ → Jenkins → Fission, secret management with Hashicorp Vault, and SSO via Azure Entra ID (OIDC) — all running on Kubernetes with Istio handling traffic.

Before mCloud, I shipped an end-to-end AIOps platform: Prometheus and the ELK stack for observability, FastAPI for the agent layer, and Ansible for closed-loop remediation, reducing manual operational overhead through automated remediation workflows.
